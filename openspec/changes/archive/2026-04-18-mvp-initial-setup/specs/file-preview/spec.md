## ADDED Requirements

### Requirement: 生成ファイルをブラウザ上でプレビューできる

生成されたファイル一覧と各ファイルの内容をブラウザ上で表示する。ファイルはツリー構造またはリスト形式で一覧表示し、選択したファイルの内容をコードビューアで表示する。

#### Scenario: ファイル一覧の表示
- **WHEN** ファイル生成が完了し、プロジェクト詳細ページを表示する
- **THEN** 生成されたすべてのファイルがファイルパスとともに一覧表示される

#### Scenario: ファイル内容のプレビュー
- **WHEN** ユーザーがファイル一覧からファイルを選択する
- **THEN** そのファイルの内容がMarkdownまたはコードとしてシンタックスハイライト付きで表示される

### Requirement: 生成ファイルを簡易編集できる

ユーザーは生成されたファイルの内容をブラウザ上のテキストエディタで編集・保存できる。

#### Scenario: ファイルの編集
- **WHEN** ユーザーがプレビュー画面で「編集」ボタンを押す
- **THEN** テキストエディタが表示され、ファイル内容を編集できる

#### Scenario: 編集の保存
- **WHEN** ユーザーが編集内容を保存する
- **THEN** `PUT /api/v1/projects/{id}/files/{file_id}` が呼ばれ、更新された内容がgenerated_filesテーブルに保存される

#### Scenario: 編集のキャンセル
- **WHEN** ユーザーが編集をキャンセルする
- **THEN** 変更は破棄され、元の内容が表示される
